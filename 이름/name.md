## 이름

```
var d: Int                    (x)

```

> 의미 있는 단어를 다른 의미로 사용(예: accountList)

### 의미 있게 구분
```
// 연속된 숫자
n1, n2, n3...    (X)

// 불용어
var name_string = "X"

```
